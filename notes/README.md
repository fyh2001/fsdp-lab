# Notes

不绑定具体模型的长文沉淀：源码阅读笔记、原理分析、对比研究、reading list 等。

与 [`issues/`](../issues/) 的边界：

- **issues/**：被动遇到的 bug 现场（"为什么坏了？怎么修？"）
- **notes/**：主动学习的产物（"它到底是怎么工作的？"）

## 建议组织方式

```text
notes/
├── fsdp1-vs-fsdp2-internals.md
├── allgather-overlap-mechanism.md
├── activation-checkpointing-deep-dive.md
├── memory-layout-of-flat-param.md
└── reading-list.md           # 论文 / PR / 优秀博客收藏
```
